# AI Website Copy Generator for Local Businesses

## 📋 Project Overview
Generate homepage copy with clear value proposition, create service page descriptions tailored to business type, write strong CTA sections, and ensure content is simple, persuasive, and website-ready.

## 🛠️ Tools
- ChatGPT / Claude / Gemini
- Lovable / Framer AI
- Google Docs / MS Word
- GitHub

## 🎓 Skills Gained
- Prompt engineering for web content
- Website copywriting
- Conversion-focused messaging
- Client requirement understanding
- AI content workflows

## ✨ Key Features
✓ Generate homepage copy with clear value proposition
✓ Create service page descriptions tailored to business type
✓ Write strong CTA sections (contact, booking, enquiry)
✓ Adapt tone for different local businesses (salon, cafe, clinic, agency)
✓ Ensure content is simple, persuasive, and website-ready

## 📦 Deliverable
A complete AI-generated website copy set (Homepage + Services + CTA content) created using structured prompts, documented with prompt logic and outputs in this public GitHub repository.

## 🚀 Getting Started
1. Check the [Issues](../../issues) section for task breakdown
2. Review prompt templates in the `/prompts` folder
3. Store generated content in `/outputs` folder
4. Document your process and learnings
