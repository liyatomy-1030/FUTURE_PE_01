# FUTURE_PE_01 - AI Website Copy Generator for Local Businesses

## Task

Create an AI-generated website copy set for a local business using structured prompt engineering.

## Sample Client

Business name: GlowNest Salon  
Business type: Local beauty salon  
Location: Kochi, Kerala  
Audience: Working women, college students, brides-to-be, and regular salon clients  
Goal: Increase appointment bookings through clear, persuasive, website-ready copy.

## Prompt Strategy

The prompt system was designed in four steps:

1. Collect business context, target audience, services, tone, and conversion goal.
2. Generate homepage copy with a clear value proposition.
3. Generate service page descriptions with benefit-focused language.
4. Generate CTA sections for booking, enquiry, and consultation.

The output was reviewed for clarity, local business relevance, persuasive tone, and website readiness.

## Structured Prompts Used

### Prompt 1 - Business Understanding

```text
Act as a professional website copywriter for local businesses.

Create a website copy strategy for the following business:
- Business name: GlowNest Salon
- Business type: Beauty salon
- Location: Kochi, Kerala
- Target audience: Working women, students, brides-to-be, and regular salon clients
- Services: Hair styling, hair spa, bridal makeup, facials, waxing, threading, manicure, pedicure
- Brand tone: Warm, polished, friendly, trustworthy
- Website goal: Increase appointment bookings

Return:
1. Brand positioning
2. Primary audience pain points
3. Website messaging angle
4. Recommended tone
5. Main conversion goal
```

### Prompt 2 - Homepage Copy

```text
Using the business strategy above, write complete homepage copy for GlowNest Salon.

Include:
- Hero headline
- Hero subheadline
- Primary CTA
- Short trust section
- About section
- Service highlights
- Why choose us section
- Booking CTA

Keep the copy simple, persuasive, and ready to paste into a website.
```

### Prompt 3 - Service Page Copy

```text
Create service page descriptions for GlowNest Salon.

For each service, write:
- Service title
- Short description
- Benefit-focused explanation
- Suggested CTA

Services:
1. Hair Styling
2. Hair Spa
3. Bridal Makeup
4. Facials
5. Waxing and Threading
6. Manicure and Pedicure
```

### Prompt 4 - CTA Variations

```text
Generate conversion-focused CTA sections for GlowNest Salon.

Create CTAs for:
1. Appointment booking
2. Bridal consultation
3. First-time customer enquiry
4. Festival or event makeover

Each CTA should include a short headline, supporting line, and button text.
```
